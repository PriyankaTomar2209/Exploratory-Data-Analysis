# Exploratory-Data-Analysis
This is a repository with code and notebooks for Exploratory Data Analysis (EDA) and data visualization in python using various python libraries like matplotlib and seaborn.

About the Data : The "movielens_EDA" is the exloratory data analysis of the movielens dataset.
Introduction : A research group at University of Minnesota known as GroupLensResearch has collected and made available data sets from the MovieLensweb site (https://movielens.org). The data sets were collected over various periods of time, depending on the size of the set.

Data set ‘MovieLens 1M’ consists of three distinct data sets – ‘ratings’, ‘users’ and ‘movies’. Overall the data consists of 6040 users, 3952 movies and 1 million total ratings of movies rated by the users.
